# 合同会社N-ONE ランディングページ

## 1. プロジェクト概要
北海道虻田郡ニセコ町に拠点を置く合同会社N-ONEの公式ランディングページ。清掃・除雪・草刈りを中心とした地域密着型サービスを提供。地元住民や別荘オーナー向けに、信頼感と温かみのあるブランドイメージを構築する一枚LP。

## 2. ページ構成
- `/` - ランディングページ（全セクションを1ページに集約）

## 3. セクション構成
- [x] ファーストビュー（Hero）
- [x] About（会社紹介）
- [x] サービスメニュー（3大サービス）
- [x] メニュー詳細・料金（4品目グリッド）
- [x] 基本情報（会社情報カードグリッド）
- [x] お客様の声
- [x] 基本情報・アクセス・電話予約
- [x] フッター
- [x] 全セクションスクロールアニメーション

## 4. データモデル設計
今回は静的LPのため、データベース不要。

## 5. バックエンド/外部連携計画
- Supabase: 不要
- Shopify: 不要
- Stripe: 不要
- その他: なし（電話予約のみ）

## 6. 開発フェーズ計画

### フェーズ1: LP全セクション実装
- 目標: 全7セクションを実装し、デザイン性の高いLPを完成させる
- 成果物: 完全なLPページ一式
